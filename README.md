## 📊 Proyectos destacados

### 🔵 Reporte Acuerdos — Power BI

Dashboard integral de seguimiento de acuerdos de cobranzas.

**Lo que resuelve:**  
Visibilidad en tiempo real del cumplimiento de acuerdos por equipo, entidad y negociador, con comparación contra el mes anterior.

**Aspectos técnicos:**
- Modelado de datos con 9 tablas (CRM API + Google Sheets)
- Medidas DAX: cumplimiento, comparación mensual y días hábiles restantes
- Jerarquía Tipo de Mora → Equipo → Entidad
- Automatización de envío diario de capturas a canal de Bitrix24 vía Make
- Documentación técnica completa (v14)
